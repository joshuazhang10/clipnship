# clipnship
clipnship [WIP] is a Windows/iOS clipboard synchronizer.  

clipnship aims to solve problems that other clipboard synchronizers have, including:
- Difficult installation
- Security concerns/closed-source
- High friction (having to exit the current application, opening the manager, copying, then going back...)

## Tech stack
Python, FastAPI, uvicorn, Poetry, Swift/SwiftUI
